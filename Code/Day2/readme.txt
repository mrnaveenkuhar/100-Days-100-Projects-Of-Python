-------------------------------------------||Web Scraper For Extracting Data From a Website||------------------------------------------------------
100 days of code - Day 2

About the project:
This is a simple web scraper that extracts data from a website and stores it in a Html file.
You can use any website you want 
you can save cousotmized data in the Html file
you can search any element form the website and save it separately

Warning: This code is for educational purposes only. I am not responsible for any misuse of this code.

Modules used:
1. requests
2. BeautifulSoup

How to use:
1. Install the modules
    you can use reqirements.txt file to install the modules
    or you can install the modules manually
2. Run the code
3. Enter the url of the website you want to scrape
4.You can either save the whloe data in a Html file or you can save customized data in a Html file
5. You can search any element from the website and save it separately
6. after that you can save the file by giving the name of the file

How it works:
1. The code requests the website
2. The code then parses the html code of the website
3. The code then extracts the data from the website 
4. The code then stores the data in a Html file or txt file

How to run:
1. Open the terminal
2. Navigate to the folder where the code is stored
3. Type "python webscraper.py" and press enter
4. Enter the url of the website you want to scrape
